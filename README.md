# Mindful Breathing

Guided breathing meditation app built with React, TypeScript, and Vite.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy (GitHub Pages)

1. Push to `main`.
2. In GitHub repo settings, open **Pages** and set **Source** to **GitHub Actions**.
3. The workflow in `.github/workflows/deploy.yml` deploys automatically on push.
